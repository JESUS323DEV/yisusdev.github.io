<Mis Proyectos>

AtenciaHomes

Proyecto desarrollado íntegramente por mí: diseño, maquetación, estructura y adaptación responsive. Atencia Homes es una web inmobiliaria creada con un enfoque claro en limpieza visual y navegación intuitiva. Combina la base estructural de Bootstrap con estilos personalizados y un sistema adaptable para distintos dispositivos. Es un proyecto real para un cliente, cuidando tanto la presentación como la funcionalidad en cada página.

Vanessa

Proyecto en progreso y desarrollado íntegramente por mí: diseño, maquetación, estructura y adaptación responsive. La web de Vanessa está enfocada en transmitir cercanía, confianza y profesionalidad a familias que buscan apoyo psicológico y psicopedagógico para niños y adolescentes. El diseño combina una estética cálida con una estructura clara, ordenada y fácil de navegar, pensada para guiar al usuario hacia la información que necesita sin saturarle. Es un proyecto real, creado para reforzar su presencia digital y facilitar el contacto directo con las familias que acompaña en su labor diaria.

DevChuletas

DevChuletas es mi web de apuntes y recursos sobre desarrollo web. Compilo lo que voy aprendiendo en HTML, CSS, Bootstrap y JavaScript, organizado como una guía visual y práctica. Ideal para repasar rápido sin perder el estilo. Cada sección está pensada para ser clara y directa como si el código te hablara en tu propio idioma. Es mi cuaderno digital. EN PROGRESO
